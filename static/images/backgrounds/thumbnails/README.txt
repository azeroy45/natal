Coloca aquí las miniaturas de las imágenes de fondo para las cartas natales.

Nombres de archivo requeridos:
- thumb-template-1.png (Miniatura de Morado Estelar)
- thumb-template-2.png (Miniatura de Azul Noche)
- thumb-template-3.png (Miniatura de Dorado Elegante)
- thumb-template-4.png (Miniatura de Cósmico)

Estas miniaturas se mostrarán en la interfaz para seleccionar el fondo.
Se recomienda que sean versiones reducidas de las imágenes originales,
con un tamaño aproximado de 120x80 píxeles para una carga rápida.

Puedes añadir más miniaturas y actualizarlas en el archivo:
/static/config/backgrounds.json
