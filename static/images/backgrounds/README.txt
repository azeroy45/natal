Coloca aquí las imágenes de fondo para las cartas natales.

Nombres de archivo requeridos:
- template-1.png (Morado Estelar)
- template-2.png (Azul Noche)
- template-3.png (Dorado Elegante)
- template-4.png (Cósmico)

Las miniaturas para la selección en la interfaz deben colocarse en la subcarpeta "thumbnails".

Puedes añadir más imágenes y actualizarlas en el archivo:
/static/config/backgrounds.json
